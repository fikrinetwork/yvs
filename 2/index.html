FAQ

Pertanyaan seputar hosting dan layanan yang kami sediakan

Apa itu VPS?

VPS atau Virtual Private Server merupakan sebuah mesin virtual(VM) dengan OS Ubuntu Server, vps bisa digunakan sebagai web server, game server, database server dan lain sebagainya.

Apa itu Domain?

Secara awam, domain bisa dipahami sebagai nama untuk website. Misalkan nama domain untuk website YouTube adalah Youtube.com. Domain tidak hanya digunakan untuk website, domain juga digunakan untuk email seperti support@fikri.net
